About Us


What is Expense Desk?

Hi, my name is Ritik Saxena. 'Expense Desk' is a part of my portfolio projects.

Expense Desk is a web application that helps in managing day-to-day expenses based on common categories. It also helps controlling expenses by analyzing and evaluating the expenditures.

Who is this for?

'Expense Desk' is for everyone who wants to manage and analyze their expenses like Healthcare, Groceries, Travel, Entertainment, etc.

Features of Expense Desk:

  • Add/Edit/Delete Expenses
  • Sort expenses according to 'Date' or 'Price'
  • Search expense according to different parameters
  • Chart representations for Expense Analysis
  • Section for User Profile
  • Login System with Reset Password

Technologies used:

  • Serving requests and Database connectivity: Django (Python web-framework)
  • Database: Postgresql (ORM)
  • Chart: Fusionchart API
  • For front-end: HTML, CSS, and Bootstrap
